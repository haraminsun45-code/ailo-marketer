# 귤까기 마스터 정리 문서

## 개요

`귤까기 마스터`는 브라우저에서 실행되는 귤 클릭커 게임이다. 귤을 클릭하면 귤 체력이 줄고 포인트를 얻으며, 포인트로 업그레이드를 구매해 더 빠르게 귤을 깔 수 있다.

현재 구현은 정적 HTML/CSS/JS 기반이며, `src/server.js`가 `web/` 폴더를 서빙한다.

## 실행 방법

프로젝트 루트에서 서버를 실행한다.

```powershell
node src/server.js
```

브라우저에서 아래 주소를 연다.

```text
http://localhost:4173/
```

이 환경에서는 번들 Node 경로를 사용해 실행할 수도 있다.

```powershell
& "C:\Users\ksh55\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe" src/server.js
```

## 주요 파일

- `web/index.html`: 게임 화면 구조
- `web/styles.css`: 귤 모양, 업그레이드 UI, 애니메이션, 반응형 스타일
- `web/app.js`: 게임 상태, 클릭 처리, 업그레이드, 자동 클릭커, 귤 단계 로직
- `src/server.js`: 로컬 정적 서버

## 현재 게임 이름

게임 제목은 `귤까기 마스터`다.

브라우저 탭 제목과 화면의 큰 제목 모두 `귤까기 마스터`로 설정되어 있다.

## 기본 게임 규칙

- 귤을 수동 클릭하면 포인트를 얻는다.
- 포인트 획득량은 현재 `까는 양`과 같다.
- 처음 까는 양은 `1`이다.
- 귤 체력은 화면에 바로 표시된다.
- 귤 체력이 `0`이 되면 귤이 완전히 까진 것으로 처리된다.
- 귤이 다 까지면 속살이 튀어나오는 연출과 화면 진동 효과가 나온다.
- 지원되는 기기에서는 `navigator.vibrate()`도 호출된다.

## 귤 까짐 단계

귤은 체력 비율에 따라 4단계로 보인다.

- 1단계: 거의 안 까진 상태
- 2단계: 일부 껍질이 벗겨짐
- 3단계: 더 많이 벗겨짐
- 4단계: 속살이 거의 드러남

속살은 현재 두 개의 반달 모양 조각으로 귤 내부에 표시된다.

## 귤 종류

까는 양이 특정 수치에 도달하면 귤 종류가 바뀌고 체력도 증가한다.

| 조건 | 귤 종류 | 체력 |
|---|---:|---:|
| 까는 양 1 이상 | 귤 | 100 |
| 까는 양 20 이상 | 미니 귤 | 500 |
| 까는 양 35 이상 | 돌 귤 | 1200 |
| 까는 양 50 이상 | 잎 귤 | 1700 |
| 까는 양 65 이상 | 만다린 | 2500 |
| 까는 양 75 이상 | 로봇 귤 | 3200 |

잎 귤은 둥근 귤이 아니라 잎 모양 실루엣으로 보이도록 CSS가 적용되어 있다.
로봇 귤은 전선과 기계 패널이 강조된 금속 느낌 외형이다.

## 업그레이드

### 1. 껍질 까는 양 증가

- 클릭 한 번에 줄이는 체력량을 증가시킨다.
- 포인트 획득량도 같은 값으로 증가한다.
- 예: 까는 양이 `2`면 클릭당 체력 `2` 감소, 포인트 `+2`
- 비용 공식:

```js
10 + powerLevel * 8
```

### 2. 귤 클릭 딜레이 감소

- 수동 클릭의 대기시간을 줄인다.
- 초기 수동 클릭 딜레이는 `1.00초`다.
- 한 번 업그레이드할 때마다 `0.08초`씩 감소한다.
- 최소 수동 클릭 딜레이는 `0.15초`다.
- `0.15초`에 도달한 뒤 마지막 강화로 `2200`포인트를 내면 `0.05초`까지 내려간다.
- 비용 공식:

```js
15 + delayLevel * 10
```

### 3. 자동 귤 클릭커

- 자동으로 귤을 클릭해준다.
- 자동 클릭은 수동 클릭 딜레이의 영향을 받지 않는다.
- 자동 클릭도 현재 까는 양과 같은 힘을 가진다.
- 자동 클릭도 현재 까는 양과 같은 포인트를 준다.
- 첫 구매 시 `2.0초`마다 자동 클릭한다.
- 업그레이드할 때마다 자동 클릭 간격이 `0.1초`씩 줄어든다.
- 최소 자동 클릭 간격은 `0.4초`다.
- 가격은 자동 클릭커 자체 레벨 기준으로 증가한다.
- 비용 공식:

```js
(15 + autoLevel * 10) * 2
```

예시 가격:

| 자동 클릭커 구매 횟수 | 비용 |
|---:|---:|
| 1회 | 30 |
| 2회 | 50 |
| 3회 | 70 |
| 4회 | 90 |

### 4. 포인트 배수 강화

- 포인트 획득량에 배수를 적용한다.
- 수동 클릭과 자동 클릭 모두 같은 배수를 받는다.
- 단계는 `1배`에서 시작해 `2배`, `3배`, `4배`까지 오른다.
- 비용은 처음 `300`포인트이며, 구매할 때마다 `700`씩 증가한다.
- 비용 공식:

```js
300 + multiplierLevel * 700
```

예시 가격:

| 배수 강화 단계 | 비용 |
|---:|---:|
| 2배 | 300 |
| 3배 | 1000 |
| 4배 | 1700 |

## 완료 연출

귤 체력이 `0`이 되면 다음 연출이 발생한다.

- 속살이 튀어나오는 애니메이션
- 껍질 조각이 바깥으로 벌어지는 애니메이션
- 화면 흔들림
- 기기 진동 지원 시 진동
- 약 `0.72초` 뒤 새 귤로 리셋

## 현재 구현된 시각 요소

- 클릭 시 `+포인트` 팝업
- 체력 바
- 수동 클릭 쿨다운 바
- 4단계 까짐 표시 막대
- 귤, 미니 귤, 돌 귤, 잎 귤, 만다린, 로봇 귤별 외형
- 두 개의 반달 모양 속살
- 업그레이드 카드 UI
- 해금된 귤 스테이지를 직접 고르는 변경 창
